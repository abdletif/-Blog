# Blog

## Used Technologies <hr>

💅 FrontEnd : Js, Css, Bootstrap

🛠 Backend : Laravel 8

💾 Database : MySql

☁️ Deployment : Github

To run this project make sure you ran these commands

composer install

copy .env.example .env

php artisan key:generate

php artisan migrate

php artisan serve 

now you can visit localhost:8000 and see the project
